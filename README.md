# Prototyping Web UI for Train Indexer

## Technology stack

* Java 8
* Spring Boot 1.3 + ThymeLeaf
* [Nexus Admin Template](https://themeforest.net/item/nexus-responsive-admin-template/)

## Building with OpenJDK 1.8

```bash
./gradlew build
```

## Running
```bash
java -jar train-webui-0.1.0.jar 
```
